## Getting Started

To get started, run the following commands:

```bash
composer install
php -S localhost:8000
```

## Testing

To run the tests, run the following command:

```bash
composer test
```

## Building

To build the project, run the following command:

```bash
composer build
```

## Deploying

To deploy the project, run the following command:

```bash
composer deploy
```
